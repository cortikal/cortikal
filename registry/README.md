# Registry

Community-contributed architecture templates for Cortikal.

## How to Contribute a Template

1. Create a new directory under `templates/` with a descriptive slug name
2. Add an `arch.md` file following the [arch.md specification](../docs/architecture/arch-md-spec.md)
3. Add a `README.md` describing the architecture
4. Optionally add a `preview.png` screenshot
5. Submit a Pull Request

## Available Templates

| Template | Complexity | Description |
|----------|-----------|-------------|
| [web-app-basic](templates/web-app-basic/) | Simple | React + .NET API + PostgreSQL + Redis |
